# Objective-C 学习指南

欢迎来到 Objective-C 学习之旅！

## 目录结构

- `basics/` - 基础语法和概念
- `oop/` - 面向对象编程
- `memory/` - 内存管理
- `advanced/` - 高级特性
- `examples/` - 完整示例项目

## 学习路径

### 第一阶段：基础语法
1. 基本数据类型和变量
2. 控制流（if/else, for, while）
3. 函数和方法
4. 数组和字典

### 第二阶段：面向对象编程
1. 类和对象
2. 属性和方法
3. 继承和多态
4. 协议（Protocol）

### 第三阶段：内存管理
1. 引用计数（ARC）
2. 强引用和弱引用
3. 内存泄漏预防

### 第四阶段：高级特性
1. 分类（Category）
2. 扩展（Extension）
3. 块（Block）
4. 运行时（Runtime）

## 编译和运行

### 使用 GCC/Clang 编译
```bash
clang -framework Foundation example.m -o example
./example
```

### 使用 Xcode
1. 打开 Xcode
2. 创建新的 Command Line Tool 项目
3. 选择 Objective-C 语言
4. 运行项目
